# 📦 e_commerce

A simple e-commerce Flutter application.  
This project aims to demonstrate building beautiful and functional UIs with Flutter.

---


## 🚀 Features

- 🛍️ **Product listing**
- 🧾 **Product details**
- ➕ **Add to cart**
- 🛒 **Shopping cart**
- ❤️ **Favorites (coming soon)**
- 🎨 **Clean and modern UI using Flutter**
